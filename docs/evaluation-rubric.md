# Evaluation Rubric

## Per-Call Metrics

- pickup rate
- connection duration
- first response latency
- interruption count
- objection count
- CTA attempt rate
- booked-meeting rate
- follow-up generation rate

## Per-Call Quality Scores

- stage quality
- factuality
- relevance
- objection handling
- CTA timing
- overall quality

## Per-Turn Checks

- was the turn aligned to the current sales stage
- was the reply concise
- did the turn overclaim
- did the turn ask one question or too many
- did the turn move the call forward

## Failure Labels

- rambling
- weak opener
- missed objection
- unsupported claim
- poor CTA timing
- context miss
- tool timeout
- dead air / silence

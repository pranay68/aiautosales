# Call State Machine

## Lead States

```text
LEAD_CREATED
RESEARCHING
STRATEGY_READY
POLICY_CHECKED
READY_TO_CALL
DIALING
IN_CALL
CALL_COMPLETED
FOLLOWUP_GENERATED
SEQUENCE_SCHEDULED
WON
LOST
NURTURE
BLOCKED
```

## Allowed Transitions

- `LEAD_CREATED -> RESEARCHING`
- `RESEARCHING -> STRATEGY_READY`
- `STRATEGY_READY -> POLICY_CHECKED`
- `POLICY_CHECKED -> READY_TO_CALL`
- `POLICY_CHECKED -> BLOCKED`
- `READY_TO_CALL -> DIALING`
- `DIALING -> IN_CALL`
- `IN_CALL -> CALL_COMPLETED`
- `CALL_COMPLETED -> FOLLOWUP_GENERATED`
- `FOLLOWUP_GENERATED -> SEQUENCE_SCHEDULED`
- `SEQUENCE_SCHEDULED -> WON`
- `SEQUENCE_SCHEDULED -> LOST`
- `SEQUENCE_SCHEDULED -> NURTURE`

## Enforcement Rule

Only the orchestrator may perform major lead state transitions.
